
# Customer Feedback System

A simple React-based web application that allows users to submit and view customer feedback.
This project demonstrates the use of **React components, state management, form validation, and dynamic UI rendering**.

## 🚀 Features

* Submit customer feedback through a form
* Basic form validation
* View submitted feedback instantly
* Feedback displayed in clean card format
* Simple and responsive UI

## 🧾 Form Fields

The feedback form contains the following fields:

1. **Name** – User's full name
2. **Email** – User's email address
3. **Category** – Type of feedback (Bug, Suggestion, Complaint, Other)
4. **Priority** – Importance level (Low, Medium, High)
5. **Description** – Detailed explanation of the feedback

## 📋 How It Works

1. The user fills out the feedback form.
2. Form validation checks required fields.
3. On successful submission, feedback data is stored in React state.
4. The feedback is displayed immediately in the feedback dashboard.

## 🎯 Learning Objectives

This project helps in understanding:

* React functional components
* React hooks (`useState`)
* Form handling in React
* Component communication using props
* Dynamic rendering of lists
